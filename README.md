# olist_growth_and_retention_analysis
Analyzed 100,000+ real orders from Olist, Brazil's largest e-commerce  platform using PostgreSQL, Python , Power BI. Identified that late  deliveries score 67% lower in customer satisfaction, 24% of customers  generate 50% of revenue, and 4 sellers are actively damaging platform  reputation -translating raw data into 4 prioritized business actions.
